Custom Class Creator

This documentation will describe how to use, and the available features of the Custom Class Creator (CCC). The CCC is a tool allowing the creation of custom class cards and other assets, as well as add other assets to make a fully functioning mod for both GHE and FHE automatically.

Find the documentation for the CCC here: Card Creator Documentation
