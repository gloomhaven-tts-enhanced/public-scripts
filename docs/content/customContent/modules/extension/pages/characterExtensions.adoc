= Character Extensions

Extensions related to characters.

== Data types

[#characterIdentification]
=== CharacterIdentification

Information about the affected character of extension events.

[cols="1,1m,2"]
|===
| Attribute | Type | Description

| player | number | The affected player number.
| class | string | The class name of the character.
|===

== Extension points

=== `onCharacterLevelChange`

Called when a character changes its level.

[cols="1,1m,2"]
|===
| Parameter | Type | Description

include::partial$param/character.adoc[]
| level | number | The new level for the character.
|===
