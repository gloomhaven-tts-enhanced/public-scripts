Monsters

Every monster type that is present will have a sleeve where you can see its stats and can spawn more of them manually. Each monster will also have a mat for managing their ability cards.

The current stats of a monster is shown in the center of the mat. Below the stats image, the current monster level is shown according to the scenario level set on the Scenario Level Mat. You can use the +/- buttons to increase/decrease the current monster level. This will not change the level of any existing monsters, but only the monsters that are spawned after changing the level.

To spawn new monster of that type, use the Normal or Elite button on the left of the stat sheet. This will spawn a new figure above the monster mat that then can be placed anywhere on the table.

In a different portion of the play area are ability card mats for each monster type present.

You can draw additional ability cards using the Draw button here. This button is most useful for adding enemies to the Initiative Tracker when revealing a new room during a round. Alternatively there’s also a menu for this in the Enemy Interface, as well as a Game Key.

The Shuffle button moves all ability cards from the discard pile into the ability deck and shuffles it. This also happens automatically, when you end a round using the Player Interface while the discard pile contains an ability card with a shuffle symbol.
